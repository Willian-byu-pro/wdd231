CodeNova — Learn HTML & CSS from Scratch

Project planning document including site name, purpose, user personas, scenarios, colors, typography, wireframes, sitemap, and tech stack.

Site Name

CodeNova

Reason: modern and tech-oriented name, easy to remember. Represents a platform for learning web development interactively.

Suggested domains: codenova.io, codenova.dev

Site Purpose

The CodeNova website is designed to provide a practical and interactive platform for learning HTML and CSS.

  • Step-by-step tutorials, coding challenges, articles, and real projects.
  • Accessible and modern learning environment encouraging creativity.
  • Support beginner and intermediate students to master web development basics.

Target Market & Objectives

Audience: Students, beginners, career changers, self-taught learners.

  • Teach HTML and CSS clearly and practically.
  • Create an active learning community.
  • Provide challenges, examples, and real projects.
  • Ensure accessible learning for beginners and intermediates.
  • Improve SEO with educational content.

User Personas

  • Laura (19) – beginner student; prefers visual examples and exercises.
  • Pedro (25) – career changer; seeks short, clear lessons for free time.
  • Camila (28) – designer; wants to turn Figma layouts into real web pages.

User Scenarios

  • User starts a free HTML course with step-by-step instructions and practical examples.
  • Visitor reads a CSS best-practice article and applies it to their own project.
  • Student joins a weekly coding challenge, submits solutions, and sees community rankings.
  • User downloads a complete HTML + CSS template to study the code.

SEO Plan

  • Primary keywords: "learn HTML", "CSS course", "front-end tutorials", "HTML & CSS for beginners", "basic web development".
  • Optimize tutorial titles for SEO.
  • Integrate Google Analytics and Search Console.
  • Backlinks from blogs and tech forums.
  • Continuous educational content to improve domain authority.
